# Cognizant Digital Nurture — Java FSE Assignments

This repository contains the mandatory hands-on exercises for the **Cognizant Digital Nurture Java Full Stack Engineer (FSE)** program, organized by weekly masterclass sessions.

---

## Repository Structure

```
Week1/
  Design Patterns and Principles/
    SingletonPatternExample/        ← Exercise 1: Singleton Pattern
    FactoryMethodPatternExample/    ← Exercise 2: Factory Method Pattern
  Data Structures and Algorithms/
    ECommerceSearchFunction/        ← Exercise 2: E-Commerce Search
    FinancialForecasting/           ← Exercise 7: Financial Forecasting
  PL SQL Programming/
    ControlStructures/              ← Exercise 1: Control Structures
    StoredProcedures/               ← Exercise 3: Stored Procedures
  JUnit Mockito Logging/
    JUnitSetup/                     ← Exercise 1: Setting Up JUnit
    JUnitAssertions/                ← Exercise 3: Assertions in JUnit
    JUnitAAAAndFixtures/            ← Exercise 4: AAA Pattern & Fixtures
    MockitoMockingAndStubbing/      ← Mockito Exercise 1: Mocking & Stubbing
    MockitoVerifyInteractions/      ← Mockito Exercise 2: Verify Interactions
    SLF4JBasicLogging/              ← SLF4J Exercise 1: Logging Error & Warning

Week2/
  Spring Core and Maven/
    BasicSpringApp/                 ← Exercise 1: Configuring a Basic Spring App
    DependencyInjectionApp/         ← Exercise 2: Implementing Dependency Injection
    MavenProjectApp/                ← Exercise 4: Creating a Maven Project
  Spring Data JPA/
    QuickExample/                   ← Spring Data JPA Quick Example
    JPA_vs_Hibernate_Explanation.md ← Difference: JPA, Hibernate, Spring Data JPA

Week3/
  Spring REST/
    SpringWebProject/               ← HOL-1: Spring Web Project + Load Country from XML
    HelloWorldRest/                 ← HOL-2: Hello World REST + Country Service
    JwtAuthService/                 ← HOL-5: JWT Authentication Service

Extras/
  (Non-mandatory projects, tooling, and reference material)
```
